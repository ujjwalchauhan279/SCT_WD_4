# SCT_WD_4

## Project Overview  
SCT_WD_4 is a web development project built during my internship at SkillCraft Technology. It is a modern and responsive To-Do List web application created using HTML, CSS, and JavaScript.

## Why this project?  
- To practice building interactive front-end applications with real-world utility.  
- To learn task management UI/UX patterns and dynamic DOM manipulation using JavaScript.  
- To create a deployable web app that shows clean structure and design skills.  
- To gain hands-on experience with responsive layouts and user input handling.

## Built With  
- HTML 
- CSS
- JavaScript  

## Features  
- Add new tasks with details like title, due date. 
- Edit or delete existing tasks easily.  
- Mark tasks as complete / undo completion.     
- Responsive design for both mobile and desktop viewing.

## Getting Started  

### Prerequisites  
- A modern web browser (Chrome, Firefox, Edge, Safari)  
- No special setup required; you can open `index.html` to get started
